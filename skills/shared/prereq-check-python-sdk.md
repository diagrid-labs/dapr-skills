# Check Python SDK version

Run `python --version` (on Windows, retry with `powershell -Command "python --version"` if needed). Verify the output starts is at least `3.12.x`, higher versions (3.13.x, 3.14.x etc) are accepted as well. If not installed or the version is below 3.12, inform the user they need to install the [Python 3.12+ SDK](https://www.python.org/downloads/).
